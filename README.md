
## Install các package cần thiết

Sử dụng câu lệnh:

### `npm install`

## Khởi chạy dự án

Sử dụng câu lệnh:

### `npm start`

## Lưu ý

Phải di chuyển tới thư mục gốc thì với có thể chạy dự án. Thư mục gốc ở đây là "geek-up-test"
Nếu chưa di chuyển tới thư mục gốc thì mở terminal và sử dụng câu lệnh:

### `cd geek-up-test`
